# Contributing

Thanks for your interest in improving the Creative Factory Modbus Workbench!

## Getting set up

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python main.py
```

## Making changes

1. Fork the repository and create a branch for your change.
2. Keep changes focused; match the existing code style (standard PEP 8, type hints where practical).
3. Test your change by running the app and exercising the affected feature.
4. Open a pull request describing **what** changed and **why**.

## Reporting issues

When filing a bug, please include:

- What you did and what you expected to happen
- What actually happened (error text / screenshots help)
- Your OS, Python version, and whether you used TCP or RTU

## Code of conduct

Be respectful and constructive. We want this to be a welcoming project for everyone.
